Functions

  • lpf_1st_order( sword in ui, sdword in out u1, byte in time_const) return sword 

    lpr_1st_order
    
inputs:
ui - input value (measurement)
u1 - an internal variables for the filter, used to store the state of the filter.
(u1 contains the return-value << time_const)
time_const 
nr of bits that is shifted to the right.
ex: 3 = divide by 8, which takes 8 calls to this function to achieve 1 tau
(see http://en.wikipedia.org/wiki/Low-pass_filter for explanation of tau)
return: filtered value of input.

  • lpf_2nd_order(sword in ui, sdword in out u1, sdword in out u2, byte in time_const) return sword 

    lpr_2nd_order

time_const = nr of bits that is shifted to the right 
ex: 3 = divide by 8, which takes 8 calls to achieve 1 tau
(see http://en.wikipedia.org/wiki/Low-pass_filter for explanation of tau)
u1 and u2 are internal variables for the filter.
(u1 is an indication of the first order of the output)
(u2 contains the return-value << 16)
Bug: with input 0, output can wrap around to -1 (=65535)

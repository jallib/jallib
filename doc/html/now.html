now
Now, initialises time and date in a PIC with an RTC

Author Eur van Andel, eur@fiwihex.nl Copyright © 2008, all rights reserved.
Adapted-by
Compiler >=2.4h

Description 

This library sets the time to the system time + 1 minute,
typically the time it takes to compile and program the PIC.
This library should be generated just before compiling by executing timeset.py
